Fontawesome

How to use

General

All icons require an <i> tag with a unique class, prefixed with icon-. To use, place the following code just about anywhere:

<i class="icon-envelope-alt"></i>
Font variant

We have 3 font variant ( square, rounded, and circled ), add variant class ( icon-square, icon-rounded, icon-circled ) in the <i> code

Square icon
<i class="icon-pencil icon-square"></i>
Rounded icon
<i class="icon-envelope-alt icon-rounded"></i>
Circled icon
<i class="icon-ban-circle icon-circled"></i>
Using size

We have 3 font size ( icon 32, icon 64, and icon 128 ), add size class ( icon-32, icon-64, icon-128 ) in the <i> code

Square icon 32
<i class="icon-pencil icon-square icon-32"></i>
Rounded icon 32
<i class="icon-envelope-alt icon-rounded icon-32"></i>
Circled icon 32
<i class="icon-ban-circle icon-circled icon-32"></i>
Square icon 64
<i class="icon-pencil icon-square icon-64"></i>
Rounded icon 64
<i class="icon-envelope-alt icon-rounded icon-64"></i>
Circled icon 64
<i class="icon-ban-circle icon-circled icon-64"></i>
Square icon 128
<i class="icon-pencil icon-square icon-128"></i>
Rounded icon 128
<i class="icon-envelope-alt icon-rounded icon-128"></i>
Circled icon 128
<i class="icon-ban-circle icon-circled icon-128"></i>
